# How to Run
## Tech Stack

- Three.js
- TypeScript
- Vite

## CLI Commands

Installation

```bash
npm i
```

Run dev mode

```bash
npm run dev
```

Build

```bash
npm run build
```

Run build

```bash
npm run preview
```

Test Start
```bash
npm vite
```
(if you install vite to local directory you need to use npx)
